\section{Introducción}
\label{sec:introduccion}

El objetivo de esta práctica es diseñar un sistema basado en Procesamiento de Lenguaje Natural (PLN) capaz de automatizar la categorización de movimientos bancarios a partir de una breve descripción. El sistema busca interpretar descripciones textuales para asignar cada transacción a su área de gasto correspondiente de forma precisa.

El modelo se entrena con un dataset unificado que recoge varias formas de redactar los movimientos bancarios, asegurando que el sistema sea capaz de interpretar variaciones en las descripciones, aunque más adelante el modelo se reentrenará con las descripciones del usuario haciendo que el sistema sea más preciso.
