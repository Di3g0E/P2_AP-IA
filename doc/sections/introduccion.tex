\section{Introducción}
\label{sec:introduccion}

En esta sección se debe realizar un resumen explicativo de la práctica. Se deben describir los objetivos principales, como la automatización de la categorización de movimientos bancarios utilizando técnicas de Procesamiento de Lenguaje Natural (PLN) y aprendizaje automático.

El objetivo de esta práctica es diseñar un sistema basado en Procesamiento de Lenguaje Natural (PLN) capaz de automatizar la categorización de movimientos bancarios. El sistema busca interpretar descripciones textuales para asignar cada transacción a su área de gasto correspondiente de forma precisa.

El modelo se entrena con un dataset unificado que recoge varias formas de redactar los movimientos bancarios, asegurando que el sistema sea capaz de interpretar variaciones en las descripciones.
